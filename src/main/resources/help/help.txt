THE "help" COMMAND:

Provide a simple guide for users.

HOW TO USE:
"help" - provide a list of all the commands and a brief description of what they do.

"help <command>" - provide detailed description of the <command> specified, use this to learn more about the command and the proper syntax to use it.

EXAMPLE USE:
"help" (shows list of all commands)
"help delete" (shows detailed explanation of delete command)
