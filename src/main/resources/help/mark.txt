THE "mark" COMMAND:

Marks the task at the index specified as complete, or let the user know that it is already marked complete.

HOW TO USE:
"mark <index>" - task at index becomes/stays complete. 
*note: index refers to the position of task in list

EXAMPLE USE:
"mark 1" (assuming task at position 1 on list is currently incomplete, it will be marked as complete)
"mark 2" (assuming task at position 2 on list is currently complete, it will remain complete)
